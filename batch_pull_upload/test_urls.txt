# 批量拉取上传URL列表示例文件
# 格式：URL,MediaName,ClassId,MediaStoragePath
# 以#开头的行为注释，会被忽略
# 空行也会被忽略
# 注意：MediaName、ClassId、MediaStoragePath为可选字段，可以为空。

# 其中MediaStoragePath是媒体存储路径，以/开头，只有FileID + Path 模式的子应用可以指定存储路径。
# 如果配置里指定custom_path.enable = true时，则保持原路径，以url后的path为存储路径。MediaStoragePath则忽略。
